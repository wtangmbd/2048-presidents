# HJL 2048
Fork from [2048-presidents](https://github.com/2048-class/2048-presidents)
A Udacity version of Gabriele Cirulli's [original 2048](http://gabrielecirulli.github.io/2048/). 

The original version is a small clone of [1024](https://play.google.com/store/apps/details?id=com.veewo.a1024), based on [Saming's 2048](http://saming.fr/p/2048/) (also a clone).

### Notes for the this version
Made this version for fun. Recommend to fork [the Udacity version if necessary](https://github.com/2048-class/2048-presidents). You can [fork the original repo as well](https://github.com/gabrielecirulli/2048)
